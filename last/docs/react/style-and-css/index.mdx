在 React 中，推荐将 CSS 样式写入 `.css` 文件，通过 `className` 属性引入。而不是将行内样式写在 `.js` 文件中。因为考虑性能，CSS 的 `class` 通常要**好于**行内样式。

但是有时我们需要通过状态来控制样式，例如加载动画，点赞、收藏状态，此时行内样式的重要性不言而喻。

## 状态控制样式

可以用**条件表达式**和将 `state` 或 `props` 里的字符串传入 `className`，来动态控制样式。

```css title="CSS"
.red {
  background-color: red;
}

.green {
  background-color: green;
}
```

```jsx {3} title="JavaScript"
const [lightColor, setLightColor] = useState("red");

<TrafficLight className={lightColor} />;
```

```jsx {3} title="JavaScript"
const [lightColor, setLightColor] = useState(0);

<TrafficLight className={lightColor === 0 ? "red" : "green"} />;
```
