## 安装 Vite

```bash
yarn create vite@latest
```

<p style={{ marginBottom: 8 }}>
  执行完上述命令后，可选择 <strong>Vite</strong> 提供的模版：
</p>

- vanilla
- vanilla-ts
- vue
- vue-ts
- react
- react-ts
- preact
- preact-ts
- lit
- lit-ts
- svelte
- svelte-ts
