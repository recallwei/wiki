## 将 TypeScript 编译成 JavaScript

```bash
tsc
```

## 如何运行 TypeScript 文件

运行 `.ts` 文件需要先将 **TypeScript** 编译成 **JavaScript**，然后使用 **node** 执行生成的 `.js` 文件。  
执行 `tsc` 命令编译 `.ts` 文件，再用 **node** 执行当前所在目录下的 `index.js` 文件：

```bash
tsc && node index.js
```

## 使用 Visual Studio Code 调试

### 创建调试配置文件

在 **Visual Studio Code** 侧边栏**「运行与调试」**中创建调试配置文件 `launch.json`，选择调试器 <TH>Node.js</TH>，生成如下文件：

```json title="JSON"
{
  "version": "0.2.0",
  "configurations": [
    {
      "type": "pwa-node",
      "request": "launch",
      "name": "Launch Program",
      "skipFiles": ["<node_internals>/**"],
      "program": "${workspaceFolder}/src/index.ts",
      "outFiles": ["${workspaceFolder}/**/*.js"]
    }
  ]
}
```

### 变量

在侧边栏**「运行与调试 - 变量」**中可以看到全部变量在调试过程中的值。

### 监视表达式

在侧边栏**「运行与调试 - 监视」**中添加**监视表达式**，可以根据**断点**监视该表达式的变化。
